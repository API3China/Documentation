保留参数是Airnode在[预言机集成规范 (OIS)](/ois/v1.0.0/)对象中定义的端点的一部分。 它们不映射到操作参数，被Airnode保留用于特殊目的。 请参阅 OIS文档集中的[保留参数](/ois/v1.0.0/reserved-parameters.md)文档，获得进一步了解。
